安装指南

从下载到运行的完整教程。按照本页步骤操作，即使是第一次使用也能顺利完成安装。

Step by Step

安装步骤

建议始终从官方下载页获取安装包，不要通过聊天转发或第三方渠道

1

下载安装包

前往下载页获取最新版本安装包。确认文件名包含正确的版本号。

2

验证文件完整性

前往安全校验页，使用 PowerShell 或 certutil 验证 SHA256 哈希值，确认文件未被篡改。

3

运行安装器

双击安装包启动安装向导。如果 Windows Defender 弹出提示，请先确认 SHA256 校验通过后再选择"仍要运行"。

4

配置 AI 服务商

选择你的 AI 服务商（如 OpenAI、Anthropic 等），填写 API Base URL、API Key 和模型名称。

5

选择安装目录

保持默认安装目录或选择纯英文路径。避免中文和特殊字符，减少权限和路径问题。

6

完成安装

点击开始安装，等待 Gateway 自动启动。安装完成后通过浏览器打开本地控制页面即可使用。

安装前建议

优先使用默认安装目录，减少权限问题
安装路径避免中文和特殊字符
API Key 只填在本地安装器中，不要发到群里
完成后保存访问 Token，便于后续登录
Defender 拦截时先核对 SHA256 再放行
安装过程中需要联网下载依赖组件。如果网络环境受限，请确保能访问所选 AI 服务商的 API 端点。

验证命令

PowerShell Get-FileHash "OpenClaw-Installer-Setup-1.0.0.exe" -Algorithm SHA256
CMD certutil -hashfile "OpenClaw-Installer-Setup-1.0.0.exe" SHA256

验证安装成功

安装完成后，打开 PowerShell 运行以下命令确认 Gateway 正常运行：

PowerShell openclaw status

如果看到 Gateway 状态为 running，说明安装成功。你也可以直接在浏览器打开安装完成时显示的本地地址。

PowerShell openclaw gateway status

卸载 OpenClaw

如果需要卸载，有两种方式：

方式一：Windows 设置

打开「设置 → 应用 → 已安装的应用」，搜索 OpenClaw，点击卸载。

方式二：手动删除

1. 停止 Gateway 服务

PowerShell openclaw gateway stop

2. 删除安装目录（默认在用户目录下的 .openclaw 文件夹）

卸载不会删除你的配置和工作区数据。如需完全清除，手动删除 ~/.openclaw 目录。
FAQ

常见问题

安装过程中遇到问题？这里可能有你需要的答案

Windows 提示"未识别的应用"

这是因为首版安装包尚未做代码签名。请先在安全校验页核对 SHA256，确认文件来自官方后，点击"更多信息" → "仍要运行"。

安装完成后打不开

先检查本地端口是否已启动（默认端口会在安装完成时显示），再查看安装目录中的日志文件排查具体错误。

端口冲突怎么办

如果默认端口被占用，安装器会自动尝试其他可用端口。你也可以在配置文件中手动指定端口号。

如何升级到新版本

后续版本采用覆盖安装方式，用户配置目录会保留。具体升级说明会在版本历史页面随版本发布。
